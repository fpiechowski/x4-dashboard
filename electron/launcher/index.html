Quick checklist

  • If no X4 Bridge feed appears, check that the `x4_dashboard_bridge` extension is installed and enabled in X4.
  • If button presses do not work reliably, install AutoHotkey v2 or set a custom path below.
  • If another app already uses port 3001, the launcher can attach to the existing local server instead of starting a duplicate.

How to use this

  • Use the local URL on the same PC running the server.
  • Use the LAN URL on tablets, laptops, or extra screens on the same trusted network.
  • The Lua mod should point to the host machine running this launcher.
  • Viewing the dashboard over LAN works even when remote controls stay disabled.
